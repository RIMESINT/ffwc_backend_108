# Django Apps

## data_load
This app manages the core data models for the FFWC system, including stations, water level observations, forecasts, rainfall data, and flood alerts. It provides REST API endpoints for retrieving observed and forecasted water levels, rainfall data, and flood monitoring information.

## indian_stations
Manages data related to Indian river stations, including station details and water level observations. Provides APIs for accessing transboundary river data.

## userauth
Handles user authentication, registration, and profile management. Includes JWT token handling and user station associations for both FFWC and Indian stations.

## fileuploads
Provides file upload capabilities, allowing users to upload files to the server with automatic handling of overwrites.

## app_user_mobile
Manages mobile user authentication via OTP, user profiles, and location updates. Includes FCM token handling for push notifications.

## app_bulletin
Handles agrometeorological bulletins, including listing, adding, and retrieving bulletin details and maps.

## app_crontab
Likely contains management commands or tasks for scheduled operations, such as data processing or cleanup jobs.

## app_data_deletion
Probably handles data deletion tasks or scheduled data removal.

## app_dissemination
Manages email dissemination queues for bulletins, allowing creation and management of email campaigns to mailing lists.

## app_emails
Provides dropdown APIs for email groups and mailing list management.

## app_middlewares
Contains custom middleware for the Django application, such as request/response processing or authentication middleware.

## app_subscriptions
Manages email subscriptions, allowing users to subscribe to newsletters or updates.

## app_visualization
Provides APIs for data visualization, including basin-wise forecasts, station lists, rainfall observations, stream flow forecasts, and hydrograph generation.

## app_water_watch_mobile
Enables mobile users to submit water level and rainfall observations, with APIs for station assignment, bulk data submission, and data retrieval.

## app_mobile_static_data
Serves static data for the mobile application, including professional lists, user manuals, useful links, reports, and about us information.
